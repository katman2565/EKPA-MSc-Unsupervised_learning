## 1. LeMonde_RussoUkrainian_war_articles
Tο συγκεκριμένο dataset περιλαμβάνει άρθρα από την ιστοσελίδα της εφημερίδας Le Monde σχετικά με την ρωσοουκρανική σύγκρουση. 
Τα κείμενα είναι μεταφρασμένα στα αγγλικά.

### Δομή του Dataset
Κάθε γραμμή αντιστοιχεί σε ένα δημοσιογραφικό άρθρο της Le Monde.

#### Περιγραφή 
-article_url: URL του αρχικού άρθρου

-title:	Τίτλος άρθρου

-date:	Ημερομηνία δημοσίευσης

-section:	Δημοσιογραφική ενότητα/κατηγορία του άρθρου

-lead:	Εισαγωγικό κείμενο (lead) του άρθρου

-text_eng:	Αγγλική μετάφραση του κυρίως κειμένου

-time-slot:	Σύντομη χρονική κατηγορία (tp1, tp2 κτλ.)

-site:	πηγή δεδομένων (ιστοσελίδα της Le Monde)

## 2. Le_Monde_RussoUkrainian_Comments 
Το συγκεκριμένο dataset περιλαμβάνει σχόλια και απαντήσεις χρηστών από άρθρα της Le Monde σχετικά με τη ρωσοουκρανική σύγκρουση.
Τα σχόλια είναι μεταφρασμένα στα αγγλικά.

### Δομή του Dataset
Κάθε γραμμή αντιστοιχεί είτε σε σχόλιο χρήστη είτε σε απάντηση (reply) που σχετίζεται με συγκεκριμένο άρθρο της Le Monde.

#### Περιγραφή Μεταβλητών

-article_url:	URL του αρχικού άρθρου

-article_date:	ημερομηνία δημοσίευσης του άρθρου

-text_en:	αγγλική μετάφραση του σχολίου

-comment_date: ημερομηνία δημοσίευσης του σχολίου

-comment-time:	ώρα δημοσίευσης του σχολίου

-comment-author:	ανωνυμοποιημένο username χρήστη

-comment_type:	τύπος αλληλεπίδρασης (comment ή reply)

-time_slot:	σύντομη χρονική κατηγορία (tp1, tp2 κτλ.)

-site:	γαλλική εφημερίδα Le Monde

-lang:	αρχική γλώσσα του σχολίου (FR)

-Time_period:	φάση της σύγκρουσης

-title:	τίτλος άρθρου στα γαλλικά

-title_en: αγγλική μετάφραση τίτλου άρθρου
